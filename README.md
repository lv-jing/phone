# 📞 图片电话号码提取工具

一个基于 Vue 3 和 Vite 构建的图片 OCR（光学字符识别）工具，专门用于从图片中提取电话号码。

## ✨ 功能特性

- 🖼️ **图片上传** - 支持点击上传或拖拽上传
- 📞 **电话号码识别** - 智能识别并提取图片中的电话号码
- 📱 **多种格式支持** - 支持手机号、固定电话、400/800电话等
- 📋 **一键复制** - 支持复制单个或全部电话号码
- 📊 **进度显示** - 实时显示识别进度
- 🎨 **现代化 UI** - 简洁美观的用户界面
- ✨ **智能格式化** - 自动格式化电话号码显示

## 🚀 开始使用

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📦 技术栈

- **Vue 3** - 渐进式 JavaScript 框架（Composition API）
- **Vite** - 下一代前端构建工具
- **Tesseract.js** - 纯前端的 OCR 文字识别库

## 📖 使用说明

1. 点击上传区域或拖拽图片文件到上传区域
2. 上传成功后，点击"提取电话号码"按钮
3. 等待识别完成（会显示识别进度）
4. 识别到的电话号码会以列表形式显示，可以点击"复制"按钮复制单个号码或"复制全部"复制所有号码

## 📱 支持的电话号码格式

- **手机号**：11位数字，以1开头（如：138-0013-8000）
- **固定电话**：带区号的固定电话（如：010-12345678, 0755-12345678）
- **400/800电话**：服务热线（如：400-123-4567）
- **国际格式**：带+86前缀的手机号（如：+86 13800138000）

## 📝 注意事项

- 首次使用需要下载 OCR 模型文件，可能需要一些时间
- 识别准确度取决于图片质量和电话号码清晰度
- 建议使用清晰、对比度高的图片以获得更好的识别效果
- 如果图片中没有电话号码，会显示提示信息，并可以查看原始识别文本
